Класс Item и класс Comment служат для добавления новой заявки и для добавления комментария в уже существующую заявку.
Класс Tracker служит для хранения заявок и работы с ними.
Классы AddItem, EditItem, ShowItems и ShowWithFilterItem реализуют интерфейс UserAction и служат для выполнения
действий над заявками.
Классы ConsoleInput и StubInput реализуют интерфейс Input и служат для получения ввода от пользователя.
Класс ValidaInput наследует класс ConsoleInput и позволяет обрабатывать исключения.
Класс MenuOutException служит для работы с исключительными ситуациями.
Класс MenuTracker служит для работы с меню программы.
Класс StartUI и StartUITest служат для запуска программы и запуска тестирования соответственно.
