Класс Tracker служит для общения с пользователем. Он выводит на экран приветствие и меню.
Пользователь выбирает один из пунктов меню, выполняет указания с экрана и управление
передается классу Processing.

Класс Processing нужен для обработки запросов пользователя. Он имеет методы для добавления,
редактирования, удаляения заявки и вывода заявок на экран. Для хранения всех заявок применяется
массив типа Request.

Класс Request имеет 4 поля: имя заявки, описание, дата создания и комментарии. Т.е. отдельная
заявка это отдельный экземпляр класса Request. Класс имеет конструктор для присвоения значения полям.
